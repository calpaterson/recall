Recall - bookmark your interests

Recall is a bookmarking web application that allows you to bookmark more than just web pages.

You can bookmark topics, web pages or places from any web browser, including the one on your smart phone.

Currently, Recall is in a closed, invitation-only beta. Features are still being added. Eventually it will open to the public, but in the meantime you can request an invite below.

You can browse the site as much as you like without having an invite.

Import Your Old Bookmarks From a Web Browser or Social Bookmarking Service

Many web browsers and social bookmarking services allow you to export your bookmarks as a file (in a format sometimes called "Netscape Navigator" format). You can import this file to Recall.
